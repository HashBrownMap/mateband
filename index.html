THE WEARABLE

Panic Button

Users can notify security or medical personnel whenever they feel unsafe or uncomfortable during the event and get assistance on demand.

Hydration Alerts

Via a real-time location tracking system and a compatible smartphone app, users will receive alerts on their wearable when nearing hydration stations to make sure they stay hydrated throughout the event.

THE APPLICATION

Map

A smartphone application communicates with connected beacons to locate hydration stations, both stationary and on the move.
Users are provided with a map to help them navigate to their closest station.

THE DASHBOARD

Emergency Response

An interactive user interface for swift crowd-management that is designed for on-site security and medical personnel.
Once a user activates the panic button, management will receive an alert for first response.
